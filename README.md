# AcademicStar

基于Springboot开发框架，开发学术视频发布平台小程序--学术星。实现用户注册/登录，上传视频，关注点赞，分享视频等功能。并基于SSM框架，搭建后台管理系统，实现背景音乐的控制，以及屏蔽被举报用户上传视频的功能。并通过zookeeper组件，同步小程序后台和管理后台。

环境参数：maven-3.6.0
         springboot-1.5.12
         redis-4.0.13
         durid-1.1.0
         mybatis-1.3.4
         swagger2-2.4.0
         zookeeper-3.4.11
         MySQL-5.7
         spring-4.3.8
         Pagehelper-5.1.2
